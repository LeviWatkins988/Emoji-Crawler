# Emoji Crawler Game
This program plays a little dungeon crawler type game inside of the console, using emojis as assets.

## How to Run
### In a terminal

1. Go to the project root directory.

2. Run the program.
```
   python main.py
```

### In Google Colab

1. Paste the code in main.py into a code cell.  After the previous code cell finishes, run this code cell.
